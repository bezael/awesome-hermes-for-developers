# Awesome Hermes for Developers

*[Leer en español](README.md)*

A curated catalog of [Skills](https://agentskills.io) for [Hermes Agent](https://github.com/NousResearch/hermes-agent), organized by tech stack — not by generic use case.

> Pilot: 3 of 16 planned categories (Next.js, Docker, Claude Code). The rest will be added once the format is validated.

## How this repo works

For each category you'll find:

- **Recommended skills** — real repos, verified as of the time of writing (they exist, have a license, and haven't been abandoned for years)
- **Maturity level** — stars, last update, license. This is *not* a quality rating: nobody on this project has tested 100% of every skill's code yet
- **Use cases** — what it solves, according to what the skill itself claims
- **Installation** — the actual Hermes command to install it
- **Demo video** — script ready to record (not yet recorded)
- **Real-world example** — explicitly marked as pending until someone actually runs it

## Honesty policy

- We don't make up commands, skill names, or execution results.
- If something hasn't been verified (an untested skill, an unrecorded video), we say so — we don't pad it to look complete.
- Installing any skill from a third-party repo carries the same risk as installing any open-source dependency: review the code before giving it access to your server. Hermes' own Hub marks these repos with a `community` trust level.

## Origin

This catalog originated from the free [Hermes Agent kit at Dominicode](https://www.dominicode.com/hermes-agent) — the CLI command cheat sheet and hardening config for deploying it on your VPS.
